# Frontend (Next.js)

## Requisitos
- Node.js 20+

## Inicio rápido

```bash
cp .env.local.example .env.local
npm install
npm run dev
```

Abre `http://localhost:3000`.

## Nota para GitHub

Sube todo el contenido de esta carpeta excepto `node_modules/` y `.next/`.
Eso permite que cualquier miembro del equipo ejecute `npm install` y `npm run dev`.
